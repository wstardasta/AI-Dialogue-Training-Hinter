# 快速开始指南

## 5分钟快速上手

### 1. 安装依赖（首次运行）

```powershell
cd C:\ProptPal-AI
npm install
```

### 2. 编译项目

```powershell
npm run compile
```

### 3. 运行插件

1. 在 Cursor 中打开 `C:\ProptPal-AI` 文件夹
2. 按 `F5` 键启动扩展开发宿主
3. 在新窗口中按 `Ctrl+Shift+P`，输入 "提示词助手: 打开面板"

### 4. 开始使用

- **查看提示词**：在面板中浏览所有可用的提示词
- **插入提示词**：点击"插入"按钮将提示词插入到编辑器
- **添加提示词**：点击"+ 添加新提示词"创建自己的提示词
- **搜索提示词**：使用搜索框快速找到需要的提示词

## 核心功能

### 📝 内置提示词

插件预装了以下类型的提示词：
- 代码优化
- 生成单元测试
- 代码审查
- 修复Bug
- 生成API文档

### 🔍 智能搜索

支持按以下方式搜索：
- 标题关键词
- 内容关键词
- 分类筛选

### 📊 使用统计

- 自动记录每个提示词的使用次数
- 按使用次数自动排序
- 热门提示词优先显示

### ➕ 自定义提示词

支持添加自定义提示词：
1. 点击"+ 添加新提示词"
2. 填写标题、内容和分类
3. 保存后即可使用

## 常用命令

| 命令 | 快捷键 | 说明 |
|------|--------|------|
| 打开提示词助手 | `Ctrl+Shift+P` → "提示词助手: 打开面板" | 打开主面板 |
| 添加新提示词 | `Ctrl+Shift+P` → "提示词助手: 添加新提示词" | 快速添加 |
| 插入提示词 | 右键菜单 → "插入提示词" | 在编辑器中插入 |

## 提示词分类

- **代码优化** - 性能优化、代码重构
- **代码生成** - 生成新代码、实现功能
- **代码审查** - 代码检查、安全审计
- **Bug修复** - 问题诊断、错误修复
- **文档编写** - API文档、注释生成
- **测试用例** - 单元测试、集成测试
- **架构设计** - 系统设计、架构规划
- **其他** - 自定义分类

## 数据存储位置

提示词数据保存在：
- Windows: `%APPDATA%\Cursor\User\globalStorage\prompt-helper\prompts.json`

你可以手动编辑这个文件来批量导入提示词。
